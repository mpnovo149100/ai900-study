# Machine Learning on Azure

> Replace this starter content with your own Word summary for this module.

## Types of machine learning

- **Supervised — regression:** predict a numeric value (e.g. price).
- **Supervised — classification:** predict a category (e.g. spam / not spam).
- **Unsupervised — clustering:** group similar items with no labels.

## Key concepts

- **Features** are the input columns; the **label** is what you predict.
- Split data into **training** and **validation** sets to measure real performance.
- **Azure Machine Learning** manages training, deployment, and monitoring.
- **Automated ML** tries many models/algorithms to find the best automatically.
